# Contributing to Jupiter Studio

We welcome all contributions! Whether you are a developer, designer, or just an AI enthusiast, there's a place for you.

## How to Contribute

1.  **Fork the Project:** Fork it to your own GitHub account.
2.  **Create a Branch:** `git checkout -b feature/amazing-feature`
3.  **Commit your Changes:** `git commit -m 'Add some amazing feature'`
4.  **Push to the Branch:** `git push origin feature/amazing-feature`
5.  **Open a Pull Request:** We will review your changes as soon as possible.

## Code Standards
- Ensure all code is typed with TypeScript.
- Follow the existing project structure.
- Run `npm run build` before submitting your PR to ensure no build errors.
